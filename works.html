FORGE FLOW

Problem

Manufacturing teams often struggle with fragmented data across production lines, making it difficult to track inventory, monitor operations, and make timely decisions.

Solution

I developed Forge Flow as a manufacturing platform with a landing page and dashboard for tracking production, inventory, and operational data. The interface focuses on making complex manufacturing activity easier to monitor, understand, and act on.

Tech Stack

UI/UX
  • Figma
Frontend
  • HTML
  • CSS
  • JavaScript
Backend
  • Node.js
  • PostgreSQL

Development Process

Started with user research and workflow mapping to understand manufacturing pain points. Designed wireframes and high-fidelity prototypes in Figma, iterating on the dashboard layout for data density and clarity. Built the frontend with a component-based approach, implemented real-time data displays, and integrated backend APIs for inventory and production tracking.

CURIO

Problem

Many learners struggle to find quality educational content in one place. Free and premium resources are often scattered across different platforms, making it difficult to discover courses, manage learning progress, and access trusted materials efficiently.

Solution

I designed and developed Curio, a modern online learning platform that brings together free and premium courses into a single, user-friendly experience. The platform focuses on simplifying course discovery, improving the learning journey, and providing an intuitive interface that encourages continuous learning and engagement.

Tech Stack

UI/UX
  • Figma
Frontend
  • HTML
  • CSS
  • JavaScript
  • React.js
  • Tailwind CSS
Backend
  • Node.js
  • Express.js
  • PostgreSQL

Development Process

Started by researching existing e-learning platforms to identify common usability challenges and opportunities for improvement. Designed user flows, wireframes, and high-fidelity prototypes in Figma with a focus on accessibility, simplicity, and engagement. Built a responsive frontend using React and Tailwind CSS, developed secure backend APIs for course management and user authentication, and optimized the platform for fast performance across desktop and mobile devices. Throughout development, the interface was refined through iterative testing to ensure a seamless learning experience.
